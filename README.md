# Agent 365 Resources

A static resource hub linking to Microsoft Agent 365 learning materials.

## What's included

Curated links organized into eight sections:

- **Get Started** — product overview, documentation, and core capabilities
- **News and Announcements** — blog posts and press from the Tech Community
- **Microsoft Ignite 2025 Sessions** — recorded breakout and lab sessions
- **Training and Workshops** — Microsoft Learn paths, Agent in a Day, and certification
- **Developer Resources** — Agent 365 SDK, CLI reference, and GitHub repositories
- **IT Admin and Governance** — Microsoft 365 Admin Center and agent registry guides
- **Security, Compliance, and Data Protection** — Purview, Defender XDR, and Entra Agent ID
- **Community and Support** — Tech Community blog, MVPs, and Frontier Preview sign-up

## Contributing
